## ğŸ“‹ Requirements

### Hardware
* **Microcontroller:** ESP8266 (NodeMCU or Wemos D1 Mini).
* **IR Components:** 940nm IR LED (with a transistor driver recommended).
* **LEDs:** WS2812B (NeoPixel) strip (specifically for the NeoPixel Gateway project).

### Software (Libraries)
Ensure the following are installed in your Arduino IDE:
1.  `IRremoteESP8266`
2.  `PubSubClient`
3.  `Adafruit_NeoPixel`
4.  `ESP8266TimerInterrupt`
